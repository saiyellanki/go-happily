# Go Happily

This repository contains a friendly motivational chatbot. 

## GitHub Pages (static)

The static single-page app is in the `docs/` folder and can be served directly by GitHub Pages from the `main` branch.

A root redirect has been added so the site is accessible at:

- https://saiyellanki.github.io/go-happily/

The direct app URL is:

- https://saiyellanki.github.io/go-happily/docs/
